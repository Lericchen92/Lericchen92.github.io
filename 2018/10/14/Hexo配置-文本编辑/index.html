之前搞的.Net博客实在是太费劲，懒得维护，干脆撤了站，注销了备案，在Github上面搞个静态站好啦。所以就盯上了Hexo这个简洁，模板还多的博客生成工具，emmm很适合懒人，很喜欢。这里就简单记录下安装过程啥的……

Hexo安装

Hexo的安装很简单，因为其基于Nodejs,所以安装顺序如下：

1、安装Nodejs和git

Nodejs: http://nodejs.cn/download/ 选择10.14lst版本下载安装；

git: https://www.git-scm.com/download/ 版本随意，开心就好。

2、安装Hexo（重装亦需要）

打开命令提示符，输入命令npm install hexo-cli -g

一切结束之后 输入hexo -v查看版本

3、配置git

首先打开git bash，输入命令 ssh-keygen -t rsa -C “你的github邮箱”，随后一路回车，生成密钥对，直到结束。

接下来打开生成的密钥id_rsa.pub（推荐使用everything查找，notepad++打开），将内容复制粘贴到Github-settings-‘SSH and GPG keys’-‘SSH keys’中。在git bash输入ssh -T git@github.com测试公钥是否添加成功。

然后需要配置git连接到你的GitHub，需要两条指令：

1、输入：git config –global user.email 你的Github邮箱；
2、输入：git config –global user.name 你的GitHub ID。
